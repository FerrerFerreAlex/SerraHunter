FEM-MAT-OO

  • FEM-MAT-OO: Finite element method in Matlab Object Oriented
  • Features:
    • Object oriented.
    • Lagrangian finite elements
    • Elasticity, cell-problem, Stokes, Poisson...
    • Density-like algorithms: projected gradient, IPOPT, MMA...
    • Level-set algorithms: Slerp, Hamilton-Jacobi, Projected-slerp
    • Filters (P1/P0/PDE), Perimeter
    • Other numerical techniques: incremental scheme, stl output files...
  • Github: https://github.com/FerrerFerreAlex/FEM-MAT-OO
