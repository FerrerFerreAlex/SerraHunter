Multi-scale

Motivation
  • Study of heterogeneities in continuum mechanics in a reasonable computational time
Macro-micro scales
  • An infititesimal point of the macroscopic domain is represented by a microscopic domain (RVE)
  • Approximate heterogeneities as best as possible by an homogenous component
Approaches
  • Variational multi-scale analysis
    • Computational mechanics approach based on physical assumptions (energy)
    • Very appropriate framework for FEM
    • Reference authors: Souza-Neto, Rodrigues, Efendeiev, Kouznetsova, ...
  • Asymptotic homogenization
    • Applied mathematics approach based on periodic homogenization
    • Very useful to understand convergence analysis
    • Reference authors (French/Italian/Russian): Sanchez-Palencia, J.L. Lions, Tartar, Allaire, Dal Maso, Kozlov ...
  • Both useful and complementary
